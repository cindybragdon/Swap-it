Bonjour à tous, voici quelque étape que vous devez faire pour run le front-end

Voici les étapes en ordre que vous devez faire.
1. Vous devez vérifier si votre projet est à jour. En haut à gauche sur votre branch master généralement,
    si vous voyez une flèche vers l'intérieur, allez cliqué sur update project. Vous pouvez passé à l'étape 3.

2. Si vous avez pas la flèche sur votre branch en haut à gauche. Allez dans le terminal et faites git pull. Vous pouvez passé à l'étape 3.

3. Vous devez vérifier si vous avez le node_modules, dans le module front-end. Vous faites l'étape 4, par la suite.

4. Si vous avez le node_modules, vous pouvez aller dans le terminal, aller dans le modules front-end : cd ./frontend
    et faire npm start. Vous pouvez passé à l'étape 8.

5. Si cela ne fonctionne pas aller à l'étape 7.

6. Si vous n'avez pas le node_modules, vous devez aller dans le terminal, aller dans le module front-end : cd ./frontend
    et faire npm install. Vous pouvez passé à l'étape 7.

7. Aller dans le terminal, dans le modules front-end : cd ./frontend et faire npm i axios react-router-dom bootstrap-icons.
    Vous pouvez passé à l'étape 8.

8. Vous devez aller dans le terminal, dans le module front-end : cd ./frontend et faire npm start.
    Si vous rencontrer encore des problèmes, veuillez contacté Karolann.
